## 📖 Conceptos de Kotlin Aprendidos

### ✅ Completados:
1. **Data Class**: Clase especial para datos (genera equals, hashCode, toString, copy automáticamente)
2. **val vs var**: Inmutable vs mutable
3. **Activity**: Pantalla en Android
4. **onCreate()**: Método llamado al crear una Activity
5. **setContentView()**: Conectar layout XML con código Kotlin
6. **R class**: Referencias a recursos

### 🔵 En aprendizaje:
- **Layouts XML**: Diseñar UI sin código
- **findViewById**: Conectar elementos XML con Kotlin

---

## 📱 Conceptos de Android Aprendidos

### ✅ Completados:
1. **AndroidManifest.xml**: Configuración de la app
2. **Res/values**: Recursos (strings, colors, themes)
3. **build.gradle.kts**: Dependencias del proyecto

## 🎓 Glosario Kotlin-Español

| Término | Definición Simple |
|---------|-------------------|
| Activity | Una pantalla de la app |
| View | Elemento UI (botón, texto, imagen) |
| Layout | Contenedor que organiza las vistas |
| Bundle | Mapa para guardar datos temporales |
| Intent | Mensaje para navegar entre pantallas |
| Parcelable | Para pasar objetos entre actividades |

### Comparación Flutter-Kotlin:
- Flutter usa widgets, Android usa Views
- Flutter tiene StatefulWidget, Android tiene Activity + ViewModel
- Flutter build.dart, Android tiene Gradle
